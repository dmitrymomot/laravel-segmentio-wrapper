# Segment.io wrapper for Laravel 4

See documentation for [Segment.io SDK](https://segment.com/docs/libraries/php/)


## Installation

This package is available via Composer:

```json
{
    "require": {
        "dmitrymomot/laravel-segmentio-wrapper": "1.*"
    }
}
```
